✏️ Markdown编辑器

📖 使用说明

快捷按钮：

  • B - 粗体
  • I - 斜体
  • S - 删除线
  • H1/H2/H3 - 标题
  • 🔗 - 插入链接
  • 🖼 - 插入图片
  • <> - 代码块
  • • - 无序列表
  • 1. - 有序列表
  • " - 引用
  • 👁 - 切换预览

支持的代码语言：

javascript, python, java, html, css, sql, php, c, cpp, bash, json, xml 等

在论坛中使用：

  1. 复制 inc/ 目录下的所有新文件到您的论坛
  2. 在发帖页面引入： 
    <!--#include file="inc/Dv_Security.asp"-->
<!--#include file="inc/Dv_Markdown.asp"-->

<link rel="stylesheet" href="inc/markdown.css">
<script src="inc/markdown.js"></script>
<script src="inc/highlight-embed.js"></script>
  3. 在textarea上添加class：class="markdown-editor"
  4. 编辑器会自动初始化

🔒 安全特性

已集成的安全防护：

  • ✅ XSS攻击防护 - 自动过滤危险HTML标签
  • ✅ SQL注入防护 - SafeSQL函数
  • ✅ CSRF防护 - Token验证机制
  • ✅ 文件上传安全 - 白名单验证
  • ✅ 内容安全过滤 - 移除script、iframe等
